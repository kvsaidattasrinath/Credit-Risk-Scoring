# Credit Risk Scoring Project

##  EDA (Exploratory Data Analysis)

Key insights:

- Dataset is highly imbalanced (more non-default cases)
- Credit amount is not a strong predictor of default
- Younger customers have higher default rates
- Shorter employment duration shows slight increase in default risk
- Engineered features (like credit-to-income ratio) are more useful

##  Objective
Predict whether a customer will default on a loan.

##  Dataset
Home Credit Default Risk Dataset (Kaggle)
